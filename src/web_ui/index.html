Virtual Modbus server

We hold the Marstek's single Modbus slot for setpoint control, so HA's existing Marstek-Modbus integration can't reach the inverter directly any more. This server re-publishes each battery's holding registers on a multiplexed endpoint — point HA at this_addon_host:<port>, unit ID per battery (see Circuits & Batteries tab). Read-only; we reject writes. restart required
